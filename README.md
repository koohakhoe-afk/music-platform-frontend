# 음원 판매 플랫폼 (React + MUI)

## 실행 방법

1. 의존성 설치  
   ```
   npm install
   ```

2. 개발 서버 실행  
   ```
   npm start
   ```

3. Vercel 배포  
   - GitHub에 push 후 Vercel에서 Import

## 주요 기능

- 상단 메뉴/로고, 음원 카드 목록
- 회원가입/로그인 (카드형 UI)
- 음원 등록(파일 업로드)
- 미리듣기/구매(아임포트 결제 연동)
- 마이페이지(구매내역/내정보)

## API 주소

- 개발용: `http://localhost:5000`
- 실제 배포 시 Render 등에서 받은 주소로 변경
